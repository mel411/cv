# CV – Développeur Web

## Description
Ce projet est un site web de CV réalisé dans le cadre de la formation Développeur Web au CEF.

Le site présente mon profil, mes compétences et mes réalisations à travers plusieurs pages reliées entre elles.

## Technologies utilisées
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Structure du site
- Accueil : présentation générale
- Services : prestations proposées
- Réalisations : projets effectués
- Blog : articles
- Contact : formulaire de contact
- Mentions légales

## Validation
Toutes les pages HTML ont été validées avec le validateur W3C.
